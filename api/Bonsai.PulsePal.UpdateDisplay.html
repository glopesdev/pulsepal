Writes an observable sequence of text strings to the Pulse Pal oLED display.

public override IObservable<string> Process(IObservable<string> source)

Parameters

source IObservable<string>

A sequence where each value represents a text string to display on the top row of the oLED display. Text must be less than 17 characters in length.

Returns

IObservable<string>

An observable sequence that is identical to the source sequence but where there is an additional side effect of writing each text string to the Pulse Pal oLED display.
